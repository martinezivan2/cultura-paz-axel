# Cultura de Paz: Igualdad y Respeto de Género

Este sitio web promueve la cultura de paz a través de la igualdad de género, el respeto y la empatía.  
Está diseñado para sensibilizar a estudiantes y comunidades sobre la importancia de eliminar la discriminación.  
Incluye secciones educativas sobre valores como el respeto, la empatía y la equidad.  
Presenta una campaña visual que refuerza el mensaje de unidad y dignidad para todos.  
Ofrece acciones concretas que los usuarios pueden aplicar en su entorno escolar y familiar.  
Contiene recursos de apoyo y contactos útiles para orientación y ayuda.  
El diseño está pensado para ser accesible, amigable y atractivo para público joven.  
Utiliza HTML y CSS con una paleta de colores alegre y elementos visuales positivos.  
Incluye enlaces a fuentes oficiales como INMUJERES, ONU Mujeres y CONAVIM.  
Ideal para proyectos escolares, campañas comunitarias o iniciativas educativas.  
Fomenta el uso de lenguaje incluyente y la denuncia de cualquier forma de violencia.  
Este proyecto busca sembrar paz desde el respeto a la diversidad y la equidad.
